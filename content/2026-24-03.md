I think I have finally found some inspiration to write into this markdown file that I prepared last year. Inside a foldersystem that was constructed for me by ChatGPT with a few commands. Something I wish I would have done myself. Probably this whole site would have looked very different then. Not the most likely outcome, but one that is the result of trial and error, struggle and new ideas. But I now have this base, a publishing machine. 

Each article is written in markdown, titled with a date and dropped into a content folder. When a new article has been dropped, I run a script called build.js.

- node build.js
- node server.js
- Server running at http://localhost:4000

 I did not name this script myself. It was the most likely name. Maybe I should rename it today. Something a bit less likely. 

The html page gets reconstructed, displaying the new article at the top. Old articles can be accessed via tags or dates, or of course by scrolling (very likely). 

You can check out the [repo](https://github.com/louisafriederike/rss_blog.git) here if you want to have a go yourself.

This code also generates a RSS feed, that can be subscribed to or wired to other sites or endpoints.
